# 🤖 Telegram Channel Guard Bot

Бот-привратник для закрытого Telegram-канала.
Если пользователь вышел из канала — он попадает в чёрный список навсегда.

---

## 📁 Структура файлов

```
tg_bot/
├── bot.py           # Основной бот
├── web_panel.py     # Веб-панель управления
├── database.py      # Работа с базой данных
├── config.py        # Настройки
├── .env             # Твои секретные данные (создай сам!)
├── .env.example     # Шаблон для .env
└── requirements.txt # Зависимости
```

---

## 🚀 Установка и запуск

### Шаг 1 — Установи Python 3.10+
Скачай с https://python.org

### Шаг 2 — Установи зависимости
```bash
pip install -r requirements.txt
```

### Шаг 3 — Создай бота в Telegram
1. Напиши @BotFather → /newbot
2. Получи токен вида: `1234567890:ABCdef...`

### Шаг 4 — Узнай ID канала
1. Добавь @userinfobot как администратора в канал
2. Он покажет ID вида `-1001234567890`

### Шаг 5 — Настрой .env файл
```bash
cp .env.example .env
# Открой .env и заполни все поля
```

### Шаг 6 — Добавь бота в канал
1. Открой свой канал → Управление → Администраторы
2. Добавь бота как администратора
3. Выдай права: **Добавление участников** + **Блокировка участников**

### Шаг 7 — Запусти бота
```bash
# В одном терминале — бот:
python bot.py

# В другом терминале — веб-панель:
python web_panel.py
```

### Шаг 8 — Открой веб-панель
Перейди по адресу: http://localhost:5000
Введи пароль из WEB_SECRET в .env

## ⚙️ Как это работает

1. Пользователь пишет боту `/start`
2. Бот проверяет — не в чёрном списке ли он
3. Если всё чисто — выдаёт одноразовую ссылку-приглашение
4. Пользователь вступает в канал
5. **Если он выходит** → бот автоматически:
   - Добавляет в чёрный список
   - Банит в канале (чтобы другие ссылки тоже не работали)
   - Уведомляет пользователя

## 🛠️ Команды для администратора (в боте)

| Команда | Действие |
|---|---|
| `/stats` | Статистика канала |
| `/unban 123456789` | Разбанить пользователя |

## ☁️ Хостинг (куда деплоить)

### Бесплатные варианты:
- **Railway** (https://railway.app) — легко, просто загрузи папку
- **Render** (https://render.com) — аналогично

### Платные (надёжнее):
- **VPS на Timeweb/Beget/Hetzner** — от ~200 руб/мес
- Запускай через `screen` или `systemd`

### На VPS через systemd:
```bash
# /etc/systemd/system/tgbot.service
[Unit]
Description=TG Channel Bot
After=network.target

[Service]
WorkingDirectory=/path/to/tg_bot
ExecStart=/usr/bin/python3 bot.py
Restart=always

[Install]
WantedBy=multi-user.target
```

## ❓ Частые вопросы

**Q: Как узнать свой Telegram ID?**
Напиши боту @userinfobot — он ответит твоим ID.

**Q: Бот не видит когда пользователь выходит?**
Убедись, что в настройках бота включены `chat_member` updates.
В `bot.py` строка `allowed_updates=["message", "chat_member"]` — это уже настроено.

**Q: Как разбанить пользователя?**
Через веб-панель (кнопка "Разбанить") или через бота: `/unban 123456789`
После этого нужно также разбанить его в Telegram вручную или через бота.
